# Jardine Lab Website

The official website for the Jardine Lab at Newcastle University.

Live at: [jardinelab.com](https://jardinelab.com)

## About

Research group website for the Jardine Lab, focused on genomics of high-risk blood and lymphoid cancer.

## Built with

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting

## Structure

- `_layouts/` — page templates
- `_includes/` — reusable components (header, nav)
- `assets/img/` — images and logo
- `index.md` — home page
- `team.md` — people page
- `research.md` — research page
- `publications.md` — publications page
- `values.md` — values page

## Editing the site

Edit the `.md` files directly and push to `main` — GitHub Actions will rebuild and deploy automatically within 1-2 minutes.
